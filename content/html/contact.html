Contact Us!

This manual is very much a work in progress, both in terms of content, and in terms of structure and format. We welcome any and all comments or suggestions to improve this content. For comments/concerns/queries/suggestions etc. related specifically to the content, that is, what we're teaching and how we're teaching it, please email the maintainer of this manual, Cole Brookson: cole.brookson@gmail.com

For comments/concerns/queries/suggestions, or bug reports related to the actual website itself, the easiest way for us to see and subsequently make changes for that type of feedback is through submitting an issue on our GitHub repository. If you don't know what GitHub is, or how to submit an "issue", then please just email, that works totally fine as well, Cole Brookson: cole.brookson@gmail.com :).

We really do want to improve this resource, so any and all feedback is well and truly welcomed!!

About the R Manual


Learn more about this project

The R Manual is a resource jointed authored by Cole Brookson, Dr. Shelby Riskin, and Dr. Jacqueline Sztepanacz. This resource is not officially affiliated with the R Programming language, but aims to help students with the steep learning curve associated with learning R for the first time.
